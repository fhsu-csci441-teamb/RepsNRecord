===========================================
    REPS N RECORD - OVERVIEW 
===========================================
This script simulates user workout activity be creating a workout, fetching all workouts, and saving to a local JSON file.

===========================================
 1. ENSURE APP IS RUNNING
===========================================
npm run dev

===========================================
2. ENSURE ALL DEPENDENCIES ARE INSTALLED 
===========================================
npm install or npm install node-fetch

===========================================
3. RUN SCRIPT
===========================================
npx ts-node backend/tests/datacollection.test.ts

===========================================
4. VERIFY OUTPUT
===========================================
The script creates a file called dataoutput.json in the same folder.
